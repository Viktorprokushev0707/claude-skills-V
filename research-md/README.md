# research-md — Скил веб-исследования для Claude Code

## Что делает

Когда говоришь Claude "исследуй X" или "найди информацию о Y" — скил запускает пошаговый процесс:
1. Уточняет формат нужного результата
2. Ищет релевантные страницы через WebSearch
3. Скачивает их содержимое через API `markdown.new`
4. Сохраняет сырые данные и готовый результат в папку `web_research/results/`

## Главная особенность

Использует сервис **markdown.new** — он скачивает любую публичную веб-страницу и конвертирует её в чистый текст (markdown). Это позволяет Claude читать реальные страницы из интернета, а не только опираться на свои знания.

```
Ты → /research-md → Claude ищет URL → markdown.new скачивает страницы → Claude анализирует → файл с результатом
```

## Как установить

Скопируй файл `SKILL.md` в папку с командами Claude Code:

```bash
cp SKILL.md ~/.claude/commands/research-md.md
```

После этого команда `/research-md` появится в Claude Code на твоём компьютере.

## Как использовать

В Claude Code напиши:
```
/research-md
```

Claude сам спросит тему и уточнит формат результата.

Или сразу с темой:
```
/research-md исследуй конкурентов Notion в 2025 году
```

## Ограничения

| Ограничение | Детали |
|-------------|--------|
| Лимит markdown.new | 500 запросов в день на один IP |
| Максимум за сеанс | 15 страниц |
| Не работает | LinkedIn, Twitter, платный контент |
| Длинные страницы | Берёт первые 5000 символов |

## Результаты сохраняются в

```
web_research/results/{тема}_{дата}/
├── raw.md      — всё скачанное (сырые данные)
└── result.md   — итоговый анализ в нужном формате
```
